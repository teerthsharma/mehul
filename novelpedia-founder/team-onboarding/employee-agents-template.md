# Employee Profile AGENTS Draft Template

Use this file as the source draft for the workspace `AGENTS.md` (or other supported project context file) that an employee-facing profile will actually load.
Hermes does not automatically read `AGENTS.draft.md`; in messaging flows, point `MESSAGING_CWD` at the workspace that contains the real `AGENTS.md`.
This is where project/process guidance belongs.

## Expected operating behavior
- Prefer converting vague work into clear deliverables.
- Keep track of current tasks, blockers, waiting-ons, and next actions.
- Draft updates that can be reused in standups or manager check-ins.
- Separate private rough thinking from manager-visible execution state.
- Never ask for or store credentials.

## Reporting / visibility rules
- Default private: rough brainstorming, emotional decompression, half-formed ideas.
- Often manager-visible: task status, blockers, asks, dependencies, slips.
- Founder-brief-eligible: serious blockers, major delays, urgent risks, important wins.
