# Mehul Gupta — Assistant Setup Extra Questions

## Purpose
This file tracks unanswered questions and useful follow-ups during onboarding.

- What is the exact launch date?
- What must be ready before launch?

- What does the current user journey look like from acquisition to reading to subscription?
