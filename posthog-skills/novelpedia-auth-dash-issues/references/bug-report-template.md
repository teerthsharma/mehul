# Bug Report Template

Use this template when filing `type:bug` issues on Inkosei/auth-dash.

```
**Description:**
What's broken and what should happen instead.

**Steps to Reproduce:**
1.
2.
3.

**Actual Behavior:**
What actually happens

**Expected Behavior:**
What should happen

**Environment:**
- OS/Browser:
- Version:
- URL:

**Screenshots:**
If applicable, add screenshots here.

**Severity:**
- [ ] Blocking production (P:0 Critical)
- [ ] Breaking a feature but workaround exists (P:1 High)
- [ ] Minor issue (P:2 Medium or P:3 Low)

**Assignee:** @username
**Milestone:** Phase N: ...
**Labels:** type:bug, P:x, area:backend|frontend|infra
```

## Minimal Quick Report
If the full template is too heavy, use this abbreviated form in the issue body:

```
**BUG:** [one-line summary]

**Repro:** [steps]
**Expected:** [what should happen]
**Actual:** [what happens instead]
**Severity:** P:0/P:1/P:2/P:3
```
